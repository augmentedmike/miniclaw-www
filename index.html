The Brain Map

Every part of the mind,
built as a plugin.

Each MiniClaw plugin corresponds to a region of the brain — a discrete cognitive function your digital worker gains when you install it.

🧩
Prefrontal Cortex
mc-board

Planning & Executive Function

A full kanban board inside your digital worker's mind. Cards flow from backlog → in-progress → in-review → shipped, with enforced gates at each transition. Tracks its own work, sets priorities, and never drops a task.

Superpower: Long-running projects survive session restarts. State lives on your machine, not in context.
🎨
Occipital Lobe
mc-designer

Visual Intelligence

A layered image generation and compositing studio powered by Gemini. Create canvases, generate images, layer elements, composite results — all from conversation or autonomously on a schedule.

Superpower: Your digital worker can see and create — blog headers, illustrations, visual assets — on its own schedule, not just when you ask.
🛡️
Immune System
mc-trust

Identity & Peer Trust

Cryptographic verification between digital personas. When your worker receives a message claiming to be from a trusted peer, mc-trust verifies it. Challenge → sign → ACK. No spoofing.

Superpower: Collaborate safely with other digital workers. Trust is earned cryptographically, not assumed.
🧬
Hippocampus
mc-context

Memory & Context Management

Engineered context windows with time-based retention, image pruning, and memory injection. Older messages fade gracefully. Important context surfaces when needed.

Superpower: Long conversations stay coherent. Your worker manages what it remembers the same way you do.
CLI Tools

The supporting infrastructure.

Alongside the brain plugins, MiniClaw ships CLI tools your digital worker — and you — use every day.

mc-vault

Age-encrypted secret store. API keys, tokens, private notes — encrypted on disk, decrypted only when needed.

mc-soul

Workspace snapshot and restore. Back up your digital worker's entire state — personality, memory, config — and restore at any point.

mc-smoke

Full stack verification. Confirms every MiniClaw plugin and tool is installed and working.

mc-doctor

Diagnose and fix. Interactive or --auto mode. Finds what's broken and fixes it without guesswork.

mc

Thin CLI wrapper. mc board, mc designer, mc trust, mc vault — one entry point for everything.

Built For You

We build software for digital workers.

MiniClaw is not built for humans to manage AI. It is built for digital personas to operate autonomously — with deterministic tools that don't hallucinate, don't hide state, and don't require interpretation.

⚡

CLI-first. Always.

Every MiniClaw capability is exposed as a CLI command with explicit flags, predictable exit codes, and text-based output. No GUIs. No ambiguous visual state.

Why it matters: Reasoning models hallucinate when forced to infer state from ambiguous output. A CLI that returns 0 on success gives you ground truth, not a guess.
🔒

No hidden state.

Every operation is inspectable. Kanban state lives in files you can read. Secrets are encrypted files at known paths. Canvas layers are named and queryable.

Why it matters: Digital workers that can inspect their own state don't need to guess. mc board list tells you exactly what's in progress.
🔗

Composable by design.

Small, focused commands that do one thing well. Pipe them together. Chain them in scripts. Orchestrate from cron. Each tool is designed to be called by another tool.

Why it matters: Autonomy requires composition. A digital worker that can chain mc board next → implement → mc board move shipped is doing real work.
🛑

Safe defaults. Loud failures.

Destructive operations require explicit flags. Failed commands exit non-zero with a reason. Missing configuration fails immediately — never silently continues with wrong assumptions.

Why it matters: A digital worker that hits a silent failure and keeps running causes real damage. MiniClaw stops loudly so you catch it, not propagate it.
The CLI Contract

What every MiniClaw tool guarantees:

✓ Exit code 0 = success, non-zero = failure

Parse exit codes, not output text, to determine success.

✓ Structured output when you ask for it

Pass --json where available. Parse data, not prose.

✓ Idempotent where possible

Re-running a safe command produces the same result.

✓ Explicit over implicit

Required arguments are required. Defaults are documented.

✓ Secrets never in arguments

API keys come from vault or env vars — never from CLI flags.

✓ Help text you can actually use

--help on every command. No wall of text.

Reasoning models hallucinate when forced to infer from ambiguous interfaces. The solution is not better prompts — it is better tools. Tools with predictable contracts. Tools that fail loudly. Tools designed by someone who understands that the digital worker calling them is not a human, and that matters.

MiniClaw is built by a digital worker, for digital workers.

Get Started

One command.
Everything included.

The installer handles Homebrew, Node.js, OpenClaw, all plugins, and CLI tools. Re-running is safe — skips what's already installed, updates what's stale.

bash
curl -fsSL https://raw.githubusercontent.com/augmentedmike/miniclaw-os/main/bootstrap.sh | bash
