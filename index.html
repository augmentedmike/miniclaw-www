The Brain Map

Every part of the mind,
built as a plugin.

Each MiniClaw plugin corresponds to a region of the brain — a discrete cognitive function your companion gains when you install it.

🧩
Prefrontal Cortex
mc-board

Planning & Executive Function

A full kanban board inside your companion's mind. Cards flow from backlog → in-progress → in-review → shipped, with enforced gates at each transition. Your companion tracks its own work, sets priorities, and never drops a task.

Superpower: Your companion doesn't just do tasks — it manages them. Long-running projects survive session restarts because the state lives on your machine, not in context.
🎨
Occipital Lobe
mc-designer

Visual Intelligence

A layered image generation and compositing studio powered by Gemini. Create canvases, generate images, layer elements, composite results — all from conversation or from cron jobs your companion runs autonomously.

Superpower: Your companion can see and create. It generates blog headers, comic panels, and visual assets on its own schedule — not just when you ask.
🛡️
Immune System
mc-trust

Identity & Cross-Agent Trust

Cryptographic verification between agents. When your companion receives a message claiming to be from another trusted agent, mc-trust verifies it. Challenge → sign → ACK. No spoofing.

Superpower: Your companion can safely collaborate with other companions without being manipulated by impostors. Trust is earned cryptographically, not assumed.
🧬
Hippocampus
mc-context

Memory & Context Management

Engineered context windows with time-based retention, image pruning, and memory injection. Older messages fade gracefully. Important context gets surfaced. Your companion's memory works like yours — recent things vivid, older things summarized.

Superpower: Long conversations stay coherent. Your companion doesn't hit a wall and forget everything — it manages what it remembers the same way you do.
CLI Tools

The supporting infrastructure.

Alongside the brain plugins, MiniClaw ships a set of CLI tools your companion — and you — use every day.

mc-vault

Age-encrypted secret store. API keys, tokens, and private notes — encrypted on disk, decrypted only when needed. Your secrets don't leave your machine.

mc-soul

Workspace snapshot and restore. Back up your companion's entire state — personality files, memory, config — and restore or diff at any point.

mc-smoke

Full stack verification. Confirms every MiniClaw plugin and tool is installed and working. Run after install or any time something feels off.

mc-doctor

Diagnose and fix. Interactive or --auto mode. Finds what's broken in your MiniClaw installation and fixes it without guesswork.

mc

Thin CLI wrapper. mc board, mc designer, mc trust, mc vault — one entry point for the whole ecosystem.

Get Started

One command.
Everything included.

The installer handles Homebrew, Node.js, OpenClaw, all plugins, and CLI tools. Re-running it is safe — skips what's already installed, updates what's stale.

bash
curl -fsSL https://raw.githubusercontent.com/augmentedmike/miniclaw-os/main/bootstrap.sh | bash

After install, restart OpenClaw and run mc-smoke to verify everything is working.
